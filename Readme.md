# 📈 Sales & Demand Forecasting for Businesses

## 🚀 Future Interns – Machine Learning Internship Task 1

### 📌 Project Overview

This project focuses on forecasting future sales using historical business data with Machine Learning techniques. The objective is to help businesses make informed decisions regarding inventory planning, staffing, promotional campaigns, and demand forecasting.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, model comparison, and future sales prediction.

---

## 🎯 Objectives

* Analyze historical sales data
* Perform data cleaning and preprocessing
* Engineer time-based features
* Train multiple Machine Learning models
* Compare model performance
* Forecast future sales
* Generate business recommendations

---

## 🗂️ Dataset

The dataset contains historical sales information with the following features:

| Feature | Description          |
| ------- | -------------------- |
| date    | Sales date           |
| store   | Store ID             |
| sales   | Sales value (Target) |
| promo   | Promotion status     |
| holiday | Holiday indicator    |

## 📈 Model Performance

| Model             |      MAE |     RMSE |  R² Score |
| ----------------- | -------: | -------: | --------: |
| Linear Regression |    11.60 |    14.13 |     0.723 |
| Random Forest     |     4.99 |     6.37 |     0.944 |
| **XGBoost**       | **4.33** | **5.48** | **0.958** |

### Best Model

**XGBoost** achieved the highest prediction accuracy and was selected as the final forecasting model.
